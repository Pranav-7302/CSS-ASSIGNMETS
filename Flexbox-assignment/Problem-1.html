1. Describe the main difference between the CSS Flexbox layout model and the CSS Grid layout model. When would you choose to one over the other.

CSS Flexbox Layout Model: CSS Grid Layout Model:
Flexbox is primarily designed for one-dimensional layouts, either as rows or columns. It's ideal for laying out items in a single direction. CSS Grid is a two-dimensional layout system that allows you to create complex, grid-based layouts with both rows and columns.
Flexbox works well when you want to distribute space and align items along a single axis, which can be horizontal or vertical. Grid is ideal for creating overall page layouts and arranging items in a grid format, such as rows and columns. You can define both row and column tracks.
It's great for creating responsive designs within a container and for aligning items with varying sizes. It's particularly useful for creating grid-like structures, such as a news feed, image gallery, or any design where elements need to be placed in specific rows and columns.
